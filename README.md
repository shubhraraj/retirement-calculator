# Retirement Calculator

A free, client-side retirement savings calculator. No sign-up, no data sent anywhere.

## Live Demo

https://[your-github-username].github.io/retirement-calculator/

## Features

- Simple mode: age, income, savings, monthly contribution
- Advanced mode: return rate, inflation rate, Social Security, 401k match, withdrawal rate
- Live updates — no submit button needed
- Retirement readiness score with color-coded progress bar
- Projected savings vs retirement target chart

## Local Development

Just open `index.html` in a browser — no build step needed.

## Deployment

1. Create a new GitHub repo named `retirement-calculator`
2. Push this folder's contents to the `main` branch
3. Go to repo Settings → Pages → Source: Deploy from branch → `main` → `/ (root)`
4. Site will be live at `https://[your-github-username].github.io/retirement-calculator/`
